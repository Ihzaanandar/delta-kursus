# 🌐 GlobalCourse - Kursus Bahasa Global

[![Status](https://img.shields.io/badge/Status-Active-brightgreen)]()
[![Tech](https://img.shields.io/badge/Tech-HTML%20%7C%20CSS%20%7C%20JS-blue)]()

**GlobalCourse** adalah platform edukasi modern yang dirancang untuk membantu pelajar dan profesional menguasai bahasa asing dengan metode yang efektif, interaktif, dan praktis. Kami menjembatani kesenjangan komunikasi global melalui pengajaran berbasis *native speaker*.

---

## ✨ Fitur Utama

-   **Responsif & Modern**: Tampilan yang optimal di berbagai perangkat (Mobile, Tablet, Desktop).
-   **Desain Elegan**: Menggunakan tipografi *Inter* dan ikon dari *Font Awesome* untuk pengalaman pengguna yang premium.
-   **Animasi Halus**: Implementasi efek *reveal* saat scroll untuk kesan interaktif.
-   **Tanpa Dependency Berat**: Dibangun menggunakan Vanilla JS dan Pure CSS (No Frameworks required).
-   **Form Pendaftaran Terintegrasi**: Memudahkan calon peserta untuk mendaftar langsung melalui website.

---

## 📚 Program Yang Tersedia

Kami menawarkan berbagai program bahasa unggulan:

| Bahasa | Durasi | Keterangan |
| :--- | :--- | :--- |
| **Inggris** | 3 Bulan | Fokus Akademik & Bisnis |
| **Jepang** | 4 Bulan | Persiapan JLPT & Percakapan |
| **Korea** | 4 Bulan | Tata Bahasa & Budaya |
| **Mandarin** | 5 Bulan | Bahasa Bisnis Masa Depan |

---

## 🛠️ Teknologi Yang Digunakan

-   **HTML5**: Struktur semantik untuk SEO dan aksesibilitas.
-   **CSS3**: Custom properties (variables), Flexbox, dan Grid Layout.
-   **JavaScript (ES6+)**: Logika pendaftaran dan animasi scroll.
-   **Google Fonts**: Inter (Modern sans-serif type).
-   **Font Awesome**: Library ikon vektor.
